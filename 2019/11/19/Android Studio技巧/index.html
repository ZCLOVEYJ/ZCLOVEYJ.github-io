Android Studio 技巧

  1. 去掉单词的拼写检查
    File->Settings->Editor->Inspections->Typo去掉勾选。

  2. 创建class文件，默认生成Header信息
    File->Settings->Editor->File And Code Templates->File Headers

  3. 去掉AndroidManifest.xml 中<Application/>标签下app is not indexable by Google 的警告，在app:build.gradle android节点下 添加

  4. adb输出日志

  5. gradlew 命令

  6. 将terminal 命令日志输出到文件

  7. 关于华为手机不打印Log日志问题do not support AFT because of no config华为默认关闭了日志打印,如果是pad 打开计算器 输入“()()2846579()()”,如果是手机输入 “*#*#2846579#*#*”,就会进入后台设置-LOG日志-LOG全打开即可
