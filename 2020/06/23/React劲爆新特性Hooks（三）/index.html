什么是Hooks

官方文档中 Hooks are a new addition in React 16.8. They let you use state and other React features without writing a class.

useState

useEffect

仅仅有状态没法实现完整的业务流程，在特定的状态节点和实际下都需要执行特定的行为，这些行为置身于组件渲染过程之外，比如绑定事件，发起异步请求（网络请求），访问DOM元素等等，统统都称为副作用。副作用最常见的调用时机是在Mount之后、Update之后，以及Unmount之前，以往我们用相应的生命周期函数来实现，现在我们用useEffect这样一个hook函数来覆盖所有的情况。那为什么一个useEffect就可以覆盖所有的场景呢？useEffect标准上是在组件每次渲染（render）之后调用，并且可以根据自定义状态决定调用或者不调用
