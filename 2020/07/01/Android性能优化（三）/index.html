内存优化

内存优化介绍

背景介绍

  • 内存是大问题但缺乏关注

    内存问题

  • 内存抖动：锯齿状、GC导致卡顿
  • 内存泄漏：可用内存减少、频繁GC
  • 内存溢出：OOM、程序异常

优化工具选择

Memory Profiler （Android Studio 自带）

  • 实时图表展示应用内存使用量
  • 识别内存泄漏、抖动等
  • 提供捕获堆转储、强制GC以及跟踪内存分配的能力
